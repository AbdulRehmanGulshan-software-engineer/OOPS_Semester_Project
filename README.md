# OOPS_Semester_Project
C++ Object-Oriented Programming (OOP) semester project simulating a Space Colony Survival game. Players manage resources, repair modules, face random hazards, and try to survive for multiple turns. Includes dynamic memory management, file save/load functionality, and interactive console gameplay.
**Technologies Used:**  
- C++ (OOP, dynamic memory allocation, file I/O)  
- Modular project structure (`include` and `src` folders)
- **Folder Structure:**
- till not decided , folder structure will follow modern project dtructure using include,src for handling multifiles
